# 风电准双周振荡分析工具 - 产品需求文档

## 1. 产品概述

### 1.1 产品名称
**WindQBWO** - 风电出力准双周振荡识别与分解系统

### 1.2 核心价值
对全年风电出力曲线进行自动化周期分析，识别7-15天区间内的强周期成分（准双周振荡 QBWO），通过现代化Web界面提供直观的数据上传、参数配置和结果可视化体验。

### 1.3 目标用户
- 风资源工程师
- 气象研究人员
- 新能源分析师
- 数据科学从业者

---

## 2. 功能需求

### 2.1 数据上传模块
| 功能 | 描述 |
|------|------|
| 文件上传 | 支持CSV、Parquet格式风电功率时间序列 |
| 格式预览 | 上传后自动解析并显示数据预览（时间范围、点数、采样率） |
| 缺失检测 | 自动检测数据缺失情况并提示 |
| 格式转换 | 将15分钟数据自动转换为小时数据 |

### 2.2 参数设置模块
| 参数 | 默认值 | 说明 |
|------|--------|------|
| 周期下界 | 7天 | 分析周期范围下限 |
| 周期上界 | 15天 | 分析周期范围上限 |
| 去趋势窗口 | 30天 | 滑动平均窗口大小 |
| EEMD试验次数 | 50 | 集合经验模态分解试验次数 |
| 噪声标准差 | 0.1 | EEMD噪声比例 |
| 显著性水平 | 0.95 | 统计检验置信度 |

### 2.3 分析流程模块
1. **预处理**：重采样、缺失值处理、去趋势
2. **频谱分析**：Welch功率谱 + AR(1)红噪声检验
3. **时频分析**：连续小波变换(CWT) + 显著性检验
4. **自适应分解**：EEMD提取准双周分量
5. **结果验证**：带通滤波交叉验证

### 2.4 可视化模块
| 图表类型 | 描述 |
|----------|------|
| 原始数据曲线 | 全年风电功率时间序列 |
| 去趋势序列 | 预处理后的波动成分 |
| 功率谱图 | 含红噪声背景与95%置信线 |
| 小波功率谱图 | 时频分析热力图，标注7天/15天参考线 |
| IMF分解图 | 各本征模态函数的周期与方差贡献 |
| QBWO分量图 | 提取的准双周振荡时间序列 |
| 对比验证图 | EEMD与带通滤波结果对比 |

## 3. 非功能需求

### 3.1 性能需求
- 单次分析处理时间 < 60秒（全年数据）
- 界面响应时间 < 2秒
- 支持最大数据量：365天 × 96点/天 = 35,040点

### 3.2 界面设计
- 现代化深色主题设计
- 科技感与专业性兼具
- 响应式布局，适配桌面端

### 3.3 技术架构
- **前端**：HTML5 + CSS3 + JavaScript（原生）
- **后端**：Python Flask
- **数据处理**：NumPy, Pandas, SciPy, PyEMD, PyCWT
- **可视化**：Plotly.js（交互式图表）
